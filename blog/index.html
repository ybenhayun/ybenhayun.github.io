  • Problem #1) Multiples of 3 and 5 (more)
    • Problem: If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23. Find the sum of all the multiples of 3 or 5 below 1000.

    The answer to this problem can be accomplished through a simple while/for loop. Set an integer equal to 3, and progressively add 3 to the integer. This number will clearly be divisible by either 3 or 5, so add it to the sum. Like this:
    int i = first;
    while (i < range){
    sum += i;
    i += first;
    }
